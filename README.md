# Simple Timer

Un temporizador simple desarrollado con Gambas3.

## Características

*   **Fácil de usar:** Interfaz limpia y minimalista para configurar temporizadores rápidamente.
*   **Alarmas personalizables:** Elige entre varios sonidos de alarma predefinidos.
*   **Ligero:** Construido sobre Gambas3, se integra perfectamente en escritorios Linux sin consumir apenas recursos.
*   **Código Abierto:** Siéntete libre de modificar, mejorar y adaptar el código a tus necesidades.

## Descarga e Instalación

Para aquellos que deseen una instalación rápida sin necesidad de compilar ni instalar el IDE de Gambas, se proporcionarán instaladores precompilados para las siguientes distribuciones:

*   **Debian/Ubuntu:** Descarga el paquete `.deb` y haz doble clic para instalarlo, o usa `sudo dpkg -i <nombre_del_paquete>.deb`.
*   **Arch Linux:** Descarga el paquete de Arch (`.pkg.tar.zst`) e instálalo con `sudo pacman -U <nombre_del_paquete>.pkg.tar.zst`.

Una vez instalado, podrás encontrar el "Simple Timer" en el menú de aplicaciones de tu sistema.

## Instalación desde Código Fuente (para desarrollo o compilación manual)

Si prefieres compilar el proyecto desde el código fuente o deseas realizar modificaciones, sigue estos pasos:

1.  Asegúrate de tener instalado el entorno de desarrollo de Gambas 3.
2.  Clona o descarga este repositorio.
3.  Abre el proyecto con el IDE de Gambas3.
4.  Haz clic en el botón "Ejecutar" del IDE para probarlo, o compila el ejecutable desde el IDE para una instalación manual.
